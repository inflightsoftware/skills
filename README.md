# Inflight Skills

Agent skills for [Inflight](https://inflight.co) — collect design feedback directly on your staging URLs.

## Add to your project

```bash
npx skills add inflightsoftware/skills
```

Commit the installed skill files so your whole team gets it automatically. Now everyone on the team can ask their agent:

> "Set up Inflight on this project so we can get design feedback"

## Available Skills

| Skill        | Description                                                                                  |
| ------------ | -------------------------------------------------------------------------------------------- |
| **inflight** | Set up the Inflight feedback widget, authenticate, and share staging URLs for design review |

## What is Inflight?

Inflight lets teams collect design feedback directly on staging deployments. Add a lightweight script tag, share a version via the CLI, and your team can leave feedback — comments, polls, vibe checks — right on the live site.
