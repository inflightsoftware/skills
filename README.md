# Inflight Skills

Agent skills for [Inflight](https://inflight.co) — collect design feedback directly on your staging URLs.

## Install

```bash
npx skills add inflightsoftware/skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| **inflight** | Add the Inflight feedback widget to any staging site and share versions for design review |

## What is Inflight?

Inflight lets teams collect design feedback directly on staging deployments. Add a lightweight script tag, share a version via the CLI, and your team can leave feedback — comments, polls, vibe checks — right on the live site.

## Quick Start

```bash
# 1. Add the skill to your agent
npx skills add inflightsoftware/skills

# 2. Ask your agent:
#    "Set up Inflight on this project so I can share it for design feedback"
```

The skill will guide your agent through installing the CLI, authenticating, adding the widget script tag, and sharing a version.
